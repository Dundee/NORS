Požadavky
---------

PHP 5.2.0 nebo vyšší s instalovaným php_gd a vypnutou direktivou register_globals.
Apache s podporou mod_rewrite a htaccess.
MySQL 5.


Instalace
---------

1) Pokud chcete NORS pouze zkopírovat na produkční server, smažte nejprve adresář 'tests'.
2) Zkopírujte souboru do DocumentRoot (nebo podadresář) Vašeho Apache serveru.
3) Otevřete umístění ve webovém prohlížeči.
4) Následujte instalační instrukce zobrazené aplikací.
