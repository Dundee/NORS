Requirements
------------

Requires PHP 5.2.0 or higher with php_gd installed and register_globals turned off.
Requires Apache with mod_rewrite and htaccess enabled.
Requires MySQL 5.


Installation
------------

1) If you just want to copy NORS to production server then first delete 'tests' directory.
2) Copy files to DocumentRoot (or subdirectory) of your Apache server.
3) Open location in your web browser.
4) Follow installation guidelines presented by NORS.
